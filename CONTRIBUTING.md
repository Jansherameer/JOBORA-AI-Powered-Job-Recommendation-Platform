# Contributing to Jobora 🚀

Thank you for your interest in contributing to Jobora! Community contributions help make this AI-powered job search platform better for everyone.

## How to Contribute

### 1. Reporting Bugs
- Use GitHub Issues to report bugs.
- Provide a clear description and steps to reproduce.

### 2. Feature Requests
- Open an issue to discuss new features before implementing them.

### 3. Pull Requests
- Fork the repository.
- Create a feature branch (`git checkout -b feature/amazing-feature`).
- Commit your changes.
- Push to the branch and open a Pull Request.

## Coding Standards
- **Frontend:** Follow React and Tailwind CSS best practices.
- **Backend:** Ensure Prisma schema changes are documented.
- **AI Service:** Maintain modularity in Python scripts.
